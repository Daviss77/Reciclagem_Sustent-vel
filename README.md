Este projeto tem como objetivo promover a conscientização ambiental e o descarte correto de resíduos. A plataforma oferece orientações sobre reciclagem, dicas de reaproveitamento e uma lista de ONGs. A proposta é facilitar o acesso à informação e incentivar práticas sustentáveis nas comunidades urbanas.
